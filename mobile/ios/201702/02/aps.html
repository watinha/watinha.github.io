APP ENGINE

Uma plataforma eficiente para criar aplicativos para dispositivos móveis e para a Web com escalonamento automático

AVALIAÇÃO GRATUITA

Crie aplicativos com escalonamento automático

O Google App Engine é uma plataforma para criação de aplicativos da Web escaláveis e back-ends móveis. Com o App Engine, você tem acesso a serviços integrados e APIs, como armazenamentos de dados do NoSQL, memcache e uma API de autenticação do usuário, comuns para a maioria dos aplicativos.

O App Engine escalonará seu aplicativo automaticamente de acordo com a quantidade de tráfego que ele recebe. Assim, você paga somente pelos recursos que usar. Basta fazer upload de seu código, e o Google gerenciará a disponibilidade de seu aplicativo. Não existem servidores para você provisionar ou manter.
